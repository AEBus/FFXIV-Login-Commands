global commands, delays, ordering, and execution logs.

## Installation
FFXIV Login Commands is a Dalamud plugin and requires FFXIVQuickLauncher to start the game.

1. Type `/xlsettings` in chat to open Dalamud Settings.
2. Open the "Experimental" tab and scroll to "Custom Plugin Repositories".
3. Paste the repo URL below and press the "+" button.
4. Press "Save and Close".
5. Open `/xlplugins`, search for FFXIV Login Commands, and click install.
